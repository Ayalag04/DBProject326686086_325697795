מערכת זו פותחה כחלק מפרויקט קורס, ומטרתה לאפשר ניהול כולל של אורחים, עובדים, הזמנות, תשלומים ופעילויות באתר נופש או מלון.
המערכת בנויה ב־Node.js עם Express בצד השרת, מחוברת למסד נתונים PostgreSQL, ומציגה ממשק ניהול מבוסס HTML ו-Bootstrap בצד הלקוח.

פונקציונליות עיקרית של המערכת:

ניהול טבלת אנשים (person):

הוספת אדם חדש למערכת.

הצגת כל האנשים הרשומים בטבלה.

עדכון פרטי אדם קיים.

מחיקת אדם, כולל טיפול בתלויות בטבלאות אחרות כמו visitor ו־employee.


ניהול הזמנות (reservations):

הצגת רשימת ההזמנות כולל סטטוס, טווח תאריכים, אורח, תשלום ופעילות.

הפעלת פרוצדורה לעדכון סטטוסים של הזמנות שפג תוקפן.

הפעלת פרוצדורה להצגת סכום התשלום הכולל לפי מזהה תשלום.

פעילויות (activities):

הצגת פעילויות עם מקומות פנויים בלבד.

שמירה על מגבלת קיבולת פעילות לפי מספר ההזמנות בפועל.

פונקציות מתקדמות:

פונקציה שבודקת האם אדם הוא עובד, אורח, שניהם או אף אחד ומדפיסה בהתאם.

פרוצדורות להצגת סכומי תשלום לפי מזהה.

שימוש בהודעות RAISE NOTICE להצגת מידע מהשרת.

הוראות התקנה והרצה:

דרישות מוקדמות:

Node.js מותקן במערכת.

PostgreSQL פועל עם מסד נתונים תקני.

הרשאות מתאימות על הטבלאות וה-sequences במסד הנתונים.

שלבים להפעלת המערכת:

להיכנס לתיקיית הפרויקט בשם: stage 5

להתקין את כל הספריות הדרושות על ידי הרצת הפקודה:
npm install

ליצור קובץ בשם db.js המכיל את פרטי ההתחברות למסד הנתונים:
יש להזין שם המשתמש, סיסמה, שם מסד נתונים ופורט.

להריץ את הקובץ server.js שמכיל את השרת הראשי של המערכת באמצעות:
node server.js

לפתוח דפדפן ולהזין את הכתובת:
http://localhost:3000
כאן יוצג ממשק הניהול של המערכת.
